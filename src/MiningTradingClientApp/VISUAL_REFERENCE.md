# Visual Reference - Expected App Appearance

This document describes what the MiningTradingClientApp looks like when built and run.

## Overall Appearance

### Color Scheme
The app uses a dark theme inspired by Robinhood:
- **Background:** Very dark gray (#1C1C1E) - almost black
- **Cards:** Medium dark gray (#2C2C2E) - lighter than background
- **Accent:** Bright green (#00C853) - signature Robinhood green
- **Text:** White for primary, light gray for secondary

### Layout Pattern
- **Top:** Page title and content area
- **Bottom:** Fixed tab bar with 3 tabs (always visible)
- **Content:** Scrollable area with cards

## Summary

The app has:
- **Dark theme** throughout (Robinhood-inspired)
- **Green accents** for all actions and positive changes
- **Card-based layout** for all content
- **Bottom tab bar** always visible
- **Large, bold typography** for prices
- **Touch-friendly** buttons (50px+)
- **Consistent spacing** and padding
- **No shadows** (flat design)
- **Native performance** on all platforms

When built and run, it looks like a professional trading app similar to Robinhood, but for minerals instead of stocks.
